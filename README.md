# News Migration

## Getting started

### dev

migration

```bash
./bin/dev.sh migrate! up
./bin/dev.sh migrate! down
```

seeding

```bash
./bin/dev.sh seed
```

reset

```bash
./bin/dev.sh reset
```


### production

migration

```bash
./bin/prod.sh migrate! up
./bin/prod.sh migrate! down
```

seeding

```bash
./bin/prod.sh seed
```

reset

```bash
./bin/prod.sh reset
```
